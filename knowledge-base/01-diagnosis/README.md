# 01 — Diagnosis

Documents covering how CIDP is identified and confirmed.

## What belongs here

- EAN/PNS 2021 diagnostic criteria (definite, probable, possible CIDP)
- Nerve conduction study (NCS) interpretation for CIDP
- EMG findings in CIDP
- CSF analysis (elevated protein)
- Nerve biopsy criteria
- Skin biopsy / intraepidermal nerve fiber density
- MRI findings (nerve root enhancement)
- CIDP variants: typical vs atypical (MADSAM, distal CIDP, pure sensory, pure motor, focal)
- Differential diagnosis: rule out MMN, GBS, CMT, diabetes-related neuropathy, POEMS
- Diagnostic odyssey resources (average time-to-diagnosis is 3–5 years)
