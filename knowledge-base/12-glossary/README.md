# 12 — Glossary

Plain-language definitions of medical terms used across all CIDP documents. This folder feeds the LLM's ability to explain jargon to patients.

## What belongs here

- One `glossary.md` file with all terms (preferred over many small files)
- Terms should be defined for a patient audience, not a medical professional
- Include the full term and common abbreviation: "Intravenous immunoglobulin (IVIG)"
- Cross-reference related terms
