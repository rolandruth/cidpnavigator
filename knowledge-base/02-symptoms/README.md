# 02 — Symptoms

Documents describing how CIDP presents and progresses.

## What belongs here

- Typical symptom patterns: bilateral weakness, sensory loss, areflexia
- Symptom progression: relapsing-remitting vs progressive
- How symptoms differ by CIDP variant
- Fatigue in CIDP (distinct from weakness)
- Pain patterns
- Autonomic symptoms
- Cranial nerve involvement
- Functional impact: walking, hand grip, balance, ADLs
- Flare triggers (if documented in literature)
- Patient-reported outcome scales: INCAT, MRC sum score, R-ODS
